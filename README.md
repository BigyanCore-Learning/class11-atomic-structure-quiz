# Atomic Structure CBT Examination System

## Class XI Chemistry
### WBCHSE + NCERT + JEE/NEET Foundation Practice Test


## About This Project

এই project একটি professional online examination system for Class XI Chemistry students.

এই পরীক্ষাটি তৈরি করা হয়েছে:

- WBCHSE syllabus
- NCERT concepts
- JEE Main foundation level
- NEET foundation level

এর উপর ভিত্তি করে।


## Features

- 35 Atomic Structure MCQs
- Bengali + English technical terms support
- Single correct questions
- Multiple correct questions
- JEE/NEET pattern questions
- NCERT and WBCHSE aligned concepts
- Examination timer
- Question-wise time tracking
- Hint system
- Mark for Review option
- Automatic evaluation
- Topic-wise performance analysis
- Weak area prediction
- Personalized improvement suggestions
- Printable result report


# How Students Can Use This Quiz

## Student Instructions

1. Open the examination link.
2. Enter your Name and Roll Number.
3. Read all instructions carefully.
4. Start the examination.
5. Answer all questions.
6. Use Hint option if required.
7. Submit the test after completion.
8. Check your detailed performance report.

# Updating Questions

To add or modify questions, edit:


```
questions.js
```


Each question follows this format:


```javascript
{
question:"Question text",

options:[
"A option",
"B option",
"C option",
"D option"
],

answer:["A"],

hint:"Hint text",

explanation:"Explanation text",

topic:"Topic name",

difficulty:"Easy"
}
```


For multiple correct answers:

```javascript
answer:["A","C"]
```
